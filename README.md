# To-Do List Web App (Flask)

A **simple web app** to manage your daily tasks. Built with **Flask**, this app demonstrates **CRUD operations**, routing, and dynamic web pages while keeping your tasks organized in a clean, interactive interface.



## Features

- **Add new tasks** to your to-do list  
- **Update existing tasks**  
- **Delete completed tasks**  
- **Simple, interactive web interface**  
- **Built with Flask** to demonstrate CRUD operations and routing  



## How It Works

1. **Open the Web App**:  
   - Launch the app and open it in your web browser.  

2. **Add Tasks**:  
   - Use the input field to type a new task and submit it.  

3. **Edit Tasks**:  
   - Click on an existing task to update it directly.  

4. **Delete Tasks**:  
   - Click the delete button to remove completed tasks.  

5. **Dynamic Page Rendering**:  
   - All changes are reflected immediately, demonstrating Flaskâ€™s dynamic web page rendering.  



## How to Run

1. **Install Python**  
   - Make sure you have **Python 3.x** installed.  
   - Download it from [python.org](https://www.python.org/downloads/).  

2. **Install Dependencies**  
   - Navigate to your project folder and install Flask:  
     ```bash
     pip install Flask
     ```  

3. **Download the Project**  
   - Clone the repository using Git:  
     ```bash
     git clone <repository-link>
     ```  
   - Or download the ZIP file from GitHub and extract it.  

4. **Run the App**  
   - Navigate to the project folder in your terminal:  
     ```bash
     cd todo-flask-app
     ```  
   - Run the Flask app:  
     ```bash
     python app.py
     ```  
   - By default, the app runs on `http://127.0.0.1:5000/`  

5. **Open in Browser**  
   - Open your browser and go to:  
     ```
     http://127.0.0.1:5000/
     ```
