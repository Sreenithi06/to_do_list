# to_do_list
To-Do List Web App (Flask) – A simple web app to add, update, and delete tasks. Built with Flask to demonstrate CRUD operations, routing, and dynamic web pages. Helps keep daily tasks organized in a clean, interactive interface.

Features:
  1.Add new tasks to your to-do list.
  2.Update existing tasks.
  3.Delete tasks you’ve completed.
  4.Simple, interactive web interface.
  5.Built with Flask to demonstrate CRUD operations and routing.
Working
  1.Open the web app in your browser.
  2.Use the input field to add tasks.
  3.Edit tasks directly or remove them with a click.
  4.Changes are reflected immediately, showing dynamic page rendering.
